About — Image Generator

A Pipe-Works tool for AI image generation. Supports multiple diffusion models with a structured prompt composition system. The API is the single source of truth for all configuration, gallery, and favourites data.

ModelTypeResolutionHuggingFace

Prompt Structure

The compiled prompt follows a five-section composer:

Every section is optional and independent. Random variants are supported in section text using syntax like {red|blue|green}. A fresh option is chosen on each generation, including every image in a batch. 

Subject:
[Optional subject text]

Setting:
[Optional setting text]

Details:
[Optional details text]

Lighting:
[Optional lighting text]

Atmosphere:
[Optional atmosphere text]

Requirements

Image generation requires a CUDA-capable GPU with 16GB+ VRAM. Models are downloaded from HuggingFace on first use and cached locally. See the README for full system requirements and configuration options.
